# BiliComment

BiliComment是一个命令行工具，用于记录bilibili**直播**弹幕。

BiliComment is a CLI tools for recording bilibili live Danmu.

## Before use

Before using it, you need to make sure your device has Python installed.

### Install package

```shell
$ pip install ws4py
```

## Usage

```shell
$ ./BiliComment.py <roomid>
```
